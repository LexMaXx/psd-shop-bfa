# ПСД SHOP — MVP сайта библиотеки БФА

Статический прототип сайта по 4 архивам БФА.

## Запуск
Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 8000
```

## Состав
- `index.html` — главная страница
- `styles.css` — стили
- `app.js` — фильтрация и карточки
- `data/projects.json` — каталог объектов
- `assets/` — оптимизированные фото/рендеры из архивов

Исходные файлы сохранены в отдельной библиотеке `PSD_SHOP_BFA_LIBRARY`.
